# Manajemen Data Inventaris Kafe
(Portofolio Data Entry)

## Deskripsi Proyek
Proyek ini mensimulasikan pengelolaan data inventaris produk makanan dan minuman untuk sebuah kafe. Fokus utama dari proyek ini adalah melakukan entri data secara akurat, merapikan format angka ke mata uang rupiah (Rp), serta menerapkan otomatisasi untuk memudahkan kontrol stok.

## fitur & Langkah yang Diterapkan 
1. **Data Entry & Formatting:**
Memasukan 19 item menu dengan format yang rapi dan menerapkan format mata uang Rupiah (Rp) agar data seragam dan mudah dibaca.
2. **Data Validation:**
Membuat menu *dropdwon* pada kolom Status ("Tersedia" dan "Habis") untuk mencegah kesalahan input data (*human error*).
3. **Conditional Formatting:**
Memberikan warna otomatis (Hijau untuk Tersedia, Kuning untuk Habis) untuk memantau ketersediaan stok secara cepat.
4. **Formula Otomatis:**
Menggunakan fungsi 'SUM' dan 'AVERAGE' untuk menyajikan informasi total modal, total harga jual, dan rata-rata harga secara otomatis secara di bagian bawah table.
